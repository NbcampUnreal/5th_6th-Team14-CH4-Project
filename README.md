# ⚔️ Coop or Chaos (Unreal Engine / C++)

남매 캐릭터가 등장하는 협력 퍼즐 게임입니다.  
남자 캐릭터는 점프는 불가능하지만 블럭을 밀 수 있고,  
여자 캐릭터는 블럭을 밀 수 없지만 점프가 가능합니다.  

각 캐릭터는 하나의 능력만 사용할 수 있도록 제한되어 있으며,  
서로의 능력을 활용해 기믹을 해결하고 협력하지 않으면 클리어할 수 없도록 설계되었습니다.

## 🎮 게임 소개
<details>
<summary>클릭해서 보기</summary>

2명의 플레이어가 각각 남매 캐릭터를 조작하는 협력 퍼즐 게임입니다.  
각 캐릭터는 하나의 행동만 가능하도록 제한되어 있으며,  
기믹과 레벨은 두 캐릭터의 협력을 전제로 설계되었습니다.

**게임 목표:**  
각 스테이지의 퍼즐을 해결하며 앞으로 나아가 마지막 포탈에 도달하면 클리어합니다.

## 🛠 게임 기능
<details>
<summary>클릭해서 보기</summary>

### 👤🖥 캐릭터 및 UI 시스템 (조무현)

- 캐릭터 시스템<br>
  하나의 캐릭터 로직으로 여러 타입 선택 가능<br>
  향후 캐릭터 타입 확장을 고려한 구조<br>
  점프 가능 캐릭터 / 블럭 밀기 캐릭터로 분리

- 캐릭터 선택 UI<br>
  Render Texture를 활용한 캐릭터 미리보기<br>
  중복 선택 방지 기능 구현

- 인게임 UI<br>
  플레이 타임 측정 및 저장

- 가이드 UI<br>
  스테이지 진행 상황에 따른 가이드 텍스트 출력

- UI 연출<br>
  상황별 UI 표시<br>
  간단한 애니메이션 효과 적용


### 🌐 서버 및 로비 시스템 (최호석)

- 하이브리드 서버 아키텍처<br>
  로비 서버와 게임 서버를 분리한 구조로 설계<br>
  로비 서버는 매치메이킹 및 방 관리 담당<br>
  게임 서버는 실제 게임 플레이 처리

- 서버 간 통신 시스템<br>
  RESTful API 기반 서버 간 통신 구현<br>
  게임 서버 상태 및 클리어 데이터를 로비 서버로 전달<br>
  JSON 기반 데이터 송수신 및 Stateless 구조 유지

- 세션 관리 및 서버 생명주기 제어<br>
  ServerTravel을 활용한 서버 재사용 구조 구현<br>
  PlayerControllerIterator 기반 실제 접속자 수 집계 로직 구현<br>
  탈주 및 항복 플레이어 발생 시 게임 흐름 처리

- 랭킹 시스템<br>
  클리어 데이터를 기반으로 리더보드 시스템 구축<br>
  C++에서 로직 처리, 블루프린트에서 UI 표현하는 구조로 설계



### 🧩 퍼즐 기믹 시스템 (임병관)

- 최종 퍼즐 기믹<br>
  1번 방부터 6번 방까지 각각 작동 버튼 존재<br>
  랜덤으로 선택된 정답 버튼 2개를 동시에 밟아야 관문 개방

- 방 진입 퍼즐 구조<br>
  각 방 앞에 문이 배치됨<br>
  해당 번호의 방에 진입해 장애물 통과 후 버튼 상호작용으로 문 개방

- 퍼즐 진행 방식<br>
  운이 좋을 경우 최소 2개의 방만 열어도 클리어 가능<br>
  확정 클리어를 위해서는 6개의 방을 모두 개방해야 하는 구조


### 🧩 퍼즐 기믹 시스템 (차영민)

- 1번 방 기믹<br>
  좌측 방 버튼 상호작용 시 우측 방 이동 발판 생성<br>
  점프 캐릭터 기반 랜덤 유리 발판 기믹 (50:50 안전 / 추락)

- 2번 방 기믹<br>
  역할 분담을 전제로 한 협력 퍼즐 구성<br>
  점프 불가 캐릭터: 버튼 조작 및 발판 탑승<br>
  점프 가능 캐릭터: 발판 생성 및 조작

- 협력 클리어 구조<br>
  한 캐릭터가 발판을 조종<br>
  다른 캐릭터가 해당 발판을 활용해 탈출 조건 달성


### 🧩👤 퍼즐 기믹 및 모델링 시스템 (신이삭)

- 협력 퍼즐 기믹<br>
  발판 및 버튼 상호작용 기반 퍼즐 설계<br>
  캐릭터 능력 분리를 전제로 한 기믹 구성<br>
  협력하지 않으면 진행 불가한 구조

- 상호작용 블럭 기믹<br>
  블럭이 지점 A/B 사이를 상호작용에 따라 이동<br>
  이동 중 재상호작용 시 반대 지점으로 전환

- 이동형 다리 기믹<br>
  상호작용 시 3D 블럭 다리가 공중 이동<br>
  Marker 블럭 순서에 따라 경로 이동

- 캐릭터 모델링<br>
  게임에 사용되는 간단한 캐릭터 모델링 작업

</details>

## 💻 구조 및 진행 흐름
<details>
<summary>클릭해서 보기</summary>

### 1. 게임 시작
- 플레이어 닉네임 입력

### 2. 로비
- 방장이 방 생성
- 다른 플레이어가 참가하기를 통해 방에 참여

### 3. 게임 시작
- 방장이 게임 시작
- 각 플레이어가 캐릭터 선택

### 4. 게임 진행
- 스테이지별 퍼즐을 해결하며 진행
- 각 캐릭터의 능력을 활용한 협력 플레이

### 5. 게임 종료
- 마지막 포탈 도달 시 게임 클리어
- 로비로 돌아가기 버튼 선택
- 로비로 복귀

</details>
